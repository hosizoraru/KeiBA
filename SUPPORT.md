# Support

Use GitHub Issues for reproducible app problems, data parsing problems, and
UI adaptation issues.

Helpful reports include:

- Device or Mac model.
- OS version.
- App build or commit hash.
- Server selection and affected BA feature area.
- Screenshot or screen recording for UI issues.
- Console logs for crash, notification, media playback, or parser issues.

For security issues, use the reporting path in `SECURITY.md`.

For Blue Archive account, billing, or official game support, use the official
Blue Archive support channels.
